# digitalocean-example

## Requirements
* https://github.com/tfutils/tfenv
* https://docs.digitalocean.com/reference/doctl/how-to/install/

## Before first run

```bash
# https://github.com/tfutils/tfenv
tfenv install
```

```bash
# https://docs.digitalocean.com/reference/doctl/how-to/install/
doctl auth init --context <NAME>
doctl auth switch --context <NAME>
doctl account get
```
